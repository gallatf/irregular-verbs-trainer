# English Irregular Verbs Flashcards

A small web application for practicing English irregular verbs with a flashcards learning flow.

## Vision
Start with a minimal working version and improve it step by step.

## Stack
- HTML
- CSS
- JavaScript
- JSON data

## Planned learning flow
- See one verb
- Try to recall forms
- Reveal answer
- Mark known or unknown
- Continue to next card

## Main files
- `app/irregular-verbs-flashcards.html`
- `app/styles.css`
- `app/app.js`
- `data/irregular-verbs.json`

## First milestone
A simple client-side MVP with reveal, self-rating, and session counters.
