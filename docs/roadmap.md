# Roadmap

## Phase 1: MVP
- Single flashcard
- Reveal answer
- Known / unknown
- Basic counters

## Phase 2: Input mode
- User types past simple
- User types past participle
- Compare with expected answer
- Show correctness feedback

## Phase 3: Smarter practice
- Repeat unknown verbs more often
- Track difficult verbs
- Filter deck by status

## Phase 4: Better learning support
- Czech translations
- Example sentences
- Pronunciation
- Verb groups and patterns

## Phase 5: Persistence
- Save progress
- Restore previous practice state
- Export/import progress
