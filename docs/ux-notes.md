# UX notes

## UX principles
- The app should feel calm and simple.
- One main action at a time.
- The user should never wonder what to do next.
- Feedback should be immediate and clear.

## MVP screen priorities
1. Verb prompt
2. Reveal action
3. Known / Unknown actions after reveal
4. Progress counters

## Avoid
- cluttered layouts
- too many colors
- too many controls on one screen
- game-like distractions in MVP
